Luna Music Player
Luna Music Player adalah aplikasi pemutar musik berbasis desktop yang dikembangkan menggunakan Python dan berjalan secara lokal tanpa memerlukan koneksi internet. Aplikasi ini mengusung tampilan Moon Theme UI yang dirancang menggunakan Figma dan diimplementasikan dengan ttkBootstrap untuk menghasilkan antarmuka yang modern dan konsisten.

Luna mendukung pemutaran file audio MP3 dan WAV menggunakan Pygame Mixer, serta dilengkapi fitur playlist, antrian lagu, riwayat pemutaran, pencarian lagu, dan sistem login dengan peran Admin dan User. Pengelolaan data dan navigasi lagu diimplementasikan menggunakan berbagai struktur data seperti Double Linked List, Queue, Stack, Array, dan Dictionary.

Sebagai fitur tambahan, Luna menyediakan hands-free control berbasis pola tepukan (clap control) yang memungkinkan pengguna mengontrol pemutaran musik tanpa interaksi langsung dengan antarmuka. Dengan fitur yang lengkap, ringan, dan terstruktur, Luna Music Player menjadi contoh penerapan konsep struktur data dalam pengembangan aplikasi nyata.
